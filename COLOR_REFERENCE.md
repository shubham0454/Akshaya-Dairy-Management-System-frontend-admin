# Azia Admin Panel - Color Reference

## 🎨 ONLY 5 COLORS USED IN THIS ADMIN PANEL

### PRIMARY COLORS (2)
1. **Purple**: `#6F42C1`
   - CSS Variable: `--primary-purple`
   - Usage: Main brand color, primary buttons, headers, navigation

2. **Cyan**: `#007BFF`
   - CSS Variable: `--primary-cyan`
   - Usage: Secondary actions, info buttons, charts

### SUPPORTING COLORS (3)
3. **Cyan Light**: `#00CCCC`
   - CSS Variable: `--support-cyan-light`
   - Usage: Success states, positive indicators

4. **Cyan Medium**: `#0DCAF0`
   - CSS Variable: `--support-cyan-medium`
   - Usage: Additional accents, highlights

5. **Cyan Dark**: `#17A2B8`
   - CSS Variable: `--support-cyan-dark`
   - Usage: Secondary buttons, cancel actions
